## 一、项目目标

做一个**印度占星（印占 / Vedic Astrology）排盘小工具**，纯网页实现，可以：
- 本地跑：通过现有 Flask 工作台访问
- 上线跑：整个文件夹丢到 GitHub Pages / Vercel / Netlify，生成链接直接发给别人用
- 零后端依赖：所有计算都在浏览器里完成，离线也能跑

## 二、为什么做这个

1. **自媒体内容延伸**：印占 vs 西占是现成的好选题，做一个能互动的工具比单纯文章更有传播力
2. **粉丝互动钩子**：粉丝可以输入自己的出生信息直接玩，分享率更高
3. **AI 工作流练手**：作为 Claude Code + 纯前端应用的一次完整落地实践

## 三、功能范围（MVP — 第一版）

### 输入
- 出生日期（年/月/日）
- 出生时间（时/分，精确到分钟）
- 出生地点（城市名 → 自动转换经纬度，或手动输入经纬度）
- 时区（默认东八区，可调整）

### 计算
| 项 | 说明 |
|----|------|
| 行星位置 | 太阳、月亮、水星、金星、火星、木星、土星、北交点（Rahu）、南交点（Ketu） |
| 黄道制 | 恒星黄道（Sidereal），用 Lahiri Ayanamsa |
| 上升 | 根据出生时间和地点计算 Ascendant |
| 宫位系统 | Whole Sign（整宫制，印占主流） |
| 星宿 | 27 Nakshatra，基于月亮恒星经度 |

### 显示
1. **核心信息卡片**
   - 月亮星座（印占中最重要，等同于"我是什么座"）
   - 太阳星座
   - 上升星座（Lagna）
   - 月亮星宿（Nakshatra，如 Rohini、Ashwini 等）
   - 命主星（Lagna Lord）

2. **星盘图**
   - 北印度菱形 12 宫星盘（Canvas 绘制）
   - 每宫标注：宫位号、所在星座、该宫行星缩写

3. **简要解读**（可选，MVP 可暂不做）
   - 每个行星所在星座的一句话含义
   - 月亮星宿性格关键词

## 四、技术方案

### 技术栈
- **纯前端**：HTML + CSS + 原生 JavaScript（无框架）
- **无构建**：打开 index.html 就能跑，不需要 npm
- **无依赖**：所有天文计算都是自己写的 JS，不调外部 API

### 文件结构
```
vedic-astrology/
├── index.html              # 主页面，输入表单 + 结果展示
├── PLAN.md                 # 本计划文档
├── README.md               # 使用和部署说明（稍后建）
├── css/
│   └── style.css           # 暗色 + 金色主题，复用工作台风格
├── js/
│   ├── astronomy.js        # 天文基础：儒略日、行星黄经、上升点
│   ├── vedic.js            # 印占逻辑：Ayanamsa、宫位、星宿判定
│   ├── chart.js            # Canvas 绘制北印度星盘
│   ├── data.js             # 数据表：12 星座、27 星宿、9 行星属性
│   └── app.js              # 主逻辑：表单 → 计算 → 渲染
└── assets/
    └── favicon.ico         # 网站图标（可选）
```

### 关键算法决策
| 技术点 | 选择 | 理由 |
|--------|------|------|
| 行星位置 | VSOP87 简化算法（JS 移植） | 精度 ±1′ 足够印占使用，纯 JS 可实现 |
| Ayanamsa | Lahiri（Chitrapaksha） | 印度政府官方标准，最主流 |
| 宫位系统 | Whole Sign（整宫制） | 古典印占用法，实现最简单 |
| 星宿划分 | 每宿 13°20′，共 27 宿 | 标准印占方案 |

### 视觉风格
- 沿用 FP2026 工作台配色：
  - 背景：`#0b0d10`（深灰黑）
  - 卡片：`#181b21`
  - 强调色：`#c9a96e`（老钱金）
  - 文字：`#d8dce6`
- 字体：系统默认 + 思源黑体
- 风格定位：**克制、有质感、轻奢老钱**（和你设计调性一致）

## 六、验证方法

1. **基准数据**：用已知出生信息（如名人生辰）输入
2. **对比工具**：
   - [onlinejyotish.com](http://onlinejyotish.com/)
   - Jagannatha Hora（免费专业印占软件）
   - AstroSage 在线排盘
3. **误差容忍**：上升、月亮位置 ±1° 以内算通过
4. **浏览器兼容**：Chrome / Edge / 手机 Safari 都要正常显示

## 七、部署方案

| 方式 | 优缺点 | 适用场景 |
|------|--------|----------|
| **Flask 本地访问** | 本地用，方便调试 | 开发阶段 |
| **GitHub Pages** | 免费、稳定、有 GitHub 账号就行 | 长期使用，免费上线 |
| **Vercel** | 自动部署、国内访问一般 | 技术型粉丝测试 |
| **Netlify** | 免费、拖拽上传 | 快速分享 |
| **自有域名 + 阿里云 OSS** | 国内访问最快、需备案域名 | 正式对外发布 |

**推荐首选**：GitHub Pages（免费、稳定、链接永久有效）

## 九、风险与注意点

1. **精度问题**：纯 JS 实现的天文算法精度有限，需明确标注"仅供娱乐/学习参考"
2. **地名转经纬度**：MVP 阶段先手动输入经纬度，后续可接入地理编码 API
3. **时区处理**：中国用户默认东八区，但要支持海外出生的用户
4. **版权/合规**：印占涉及迷信话题，文案保持学术/文化视角，不做命理断言
